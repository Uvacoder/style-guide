Typography

Primary Typeface

Helvetica Neue Regular is the primary typeface for all text elements with a fallback order of Helvetica, system default font, Arial, then sans-serif.

The quick brown fox jumped over the lazy dog.

Alternative Typeface

Rock Salt is the alternative typface used for blockquotes and stand-out typography that should call attention to itself. Alternative typeface should be used sparingly and only to call attention to a specific passage or area. Alternative typeface should only be used in a block element that has the full width of its container, not to exceed the normal content width. Centering the text is preferred.

Line Heights

Line height is a calculation of the font size plus eight pixels. There are a few exceptions to this rule:

  • Blockquotes have a line height of their font size plus twenty pixels.
  • List items have a line height of their font size plus six pixels.

Text Colors

For readability and accessibility, all text color should have a color contrast ratio of at least 3.0. Color classes are set with default text colors that meet this guideline.
